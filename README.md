Overview

This project demonstrates a full sales analytics workflow:
- Generating a synthetic dataset of 5 million rows using Python
- Cleaning, transforming, and aggregating data
- Visualizing business insights using a Power BI dashboard

It showcases skills in data engineering, data analysis, and business intelligence.

Structure

Python Scripts
1.	generate_sales_data.py 
o	Generates a synthetic sales dataset 
o	Randomized products, categories, regions, and order dates 
2.	sales_analysis.py 
o	Cleans and preprocesses the data 
o	Creates new features (Year, Month, MonthName, OrderValue) 
o	Aggregates data for: 
	Monthly revenue 
	Top products 
	Regional sales 
o	Exports CSVs for use in Power BI
Power BI Dashboard
•	Interactive dashboard with: 
o	KPI cards (Revenue, Orders, Quantity, Avg Order Value) 
o	Monthly revenue trends 
o	Regional sales performance 
o	Top products treemap 
o	Category-wise quantity analysis
